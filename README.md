#concat-text
 Concat text helper for ember
 
 ```hbs
 {{concat-text 'a' 'b' 'c'}} => abc
 
 {{t (concat-text 'model.' v '.one')}}
 ```
 
 Add in your package.json
 
 ```javascript
  "concat-text": "0.2.0"
 ```
 
 For ember >= 1.13.0, look at http://emberjs.com/api/classes/Ember.Templates.helpers.html#method_concat
